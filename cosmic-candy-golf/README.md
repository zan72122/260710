# 🌈⛳ コズミックキャンディゴルフ

**Binary Golf 風の 3D ミニゴルフゲーム** — 4歳児が iPhone / iPad(縦画面・横画面どちらでも)で遊ぶことを想定した、おもちゃの国の玉転がしゴルフです。

Babylon.js を同梱しているので、**ビルド不要・オフラインOK**。`index.html` を開くだけで遊べます。

## あそびかた

1. `index.html` をブラウザで開く(または GitHub Pages などの静的ホスティングに置く)
2. **▶** をタップしてステージをえらぶ
3. ボールを **ゆびでひっぱって、はなす** と飛んでいきます(パチンコ式)
4. コースにいる **ターゲット(まと)** に当てて全部あつめよう
5. **のこり1つになると、そのターゲットがゴールに変身!** ボールを入れたらクリア🎉

失敗はありません。コースから落ちても、その場にやさしく戻ってきます。何回打ってもOK。

## Binary Golf 由来のルール

- ターゲットを全部集めると、**最後の1つがゴール(カップ)に変身**する
- 光っているスーパーターゲットに当てると **能力(パワーアップ)をゲット**
- ボールが止まる前に次のターゲットに当てれば **能力をチェーン**できる(コンボ⭐表示)
- 最終ステージは **ボス戦「ブロックキング」**。まわりを回る星を全部当てると、王様が巨大ゴールに変身!

## 8つの能力(スーパーターゲット)

| アイコン | 能力 | 効果 |
|---|---|---|
| 🔨 | ハンマードリル | こわせるブロックを突き破る |
| 🚀 | ロケット | 前方へ大加速 |
| 🌪️ | トルネード | 近くのターゲットへ曲がっていく |
| 🍄 | ジャンボ | ボールが巨大化、ブロックも壊せる |
| 🐢 | スロー | 時間がゆっくりになる |
| 🧲 | マグネット | ターゲットが吸い寄せられてくる |
| 🏀 | スーパーバウンス | びよんびよん跳ねる |
| 🌈 | レインボー | ぜんぶのせ(ホーミング+破壊+吸引) |

## モード・コンテンツ

- **ステージ 1〜4**: はじめてのつみき / レールのおか / こわせるブロックのまち / うごくしまとバンパー
- **ステージ 5**: ボス戦「ブロックキングのしろ」
- **♾️ エンドレスモード**: 自動生成される島をどこまでも。クリアするたび次の島へロケット移動
- **⭐ ごほうび**: ショット数で星3つ評価(甘め判定)、クリアで花火+ファンファーレ
- **🎨 きせかえ**: あつめた星でボールの模様を8種類アンロック(サッカー・てんとう虫・すいか・にくきゅう・惑星・星空・虹 など)
- **😊 遊び心**: ターゲットに顔があって当てると笑う、ボールも表情豊か(ひっぱると力み顔、放置すると居眠り💤)、コンボで画面が虹色に
- **🔊 音**: BGM・効果音はすべて Web Audio でリアルタイム生成(音声ファイルなし)。ミュートボタンあり
- **💾 自動セーブ**: 星・きせかえ・エンドレス到達島は localStorage に保存

## 4歳児向けの工夫

- 文字はほぼゼロ、すべて絵文字アイコンのUI
- 大きなボタン、大きな矢印ガイド+予測線(点線)で「ひっぱる→とぶ」が直感的
- ミスの概念なし(落下はノーカウントで復帰、打ち直し無制限)
- 縦画面・横画面の両方に自動フィット。ホーム画面に追加すればフルスクリーンで遊べます(PWA対応)

## 技術メモ

- 3D描画: **Babylon.js 7.35.0**(`vendor/babylon.min.js` に同梱、ライセンスは `vendor/babylon-license.md`)
- 物理: 幼児向けに挙動を調整した自作のころころ物理(球 vs 回転ボックスの衝突、転がり摩擦、動く床、バンパー)
- 依存パッケージ・ビルド工程なし。静的ファイルのみ
