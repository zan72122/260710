# コロコロスターゴルフ 🌟

カービィボウル（Kirby's Dream Course）にインスパイアされた、
**4歳児向けのミニゴルフゲーム**です。
iPhone / iPad の **縦画面でも横画面でも** 遊べます。

> 主人公は オリジナルキャラクターの ぷにまるちゃん（ピンクのまんまるボール）。
> ゆびで ひっぱって はなすと コロコロ ころがります。

## あそびかた

1. **ひっぱる** — 画面のどこでもいいので、ゆびで ひっぱる
2. **はなす** — ぱっと はなすと ボールが とんでいく！
3. **ポン！** — コースにいる「ちびまるちゃん」に ぜんぶ あたると…
4. **ゴール！** — キラキラのゴールあなが とうじょう。ころがして クリア！

- おっこちても だいじょうぶ。なんども やりなおせます（ゲームオーバーなし）
- フルーツを あつめると もっと たのしい 🍎
- すくない かいすうで クリアすると ★★★

## あそぶには

サーバー不要・ビルド不要の静的ページです。`index.html` をブラウザで開くだけで動きます。

- **GitHub Pages** で公開する場合: リポジトリの Pages を有効にして
  `https://<user>.github.io/<repo>/korokoro-star-golf/` にアクセス
- **ローカル確認**: このフォルダで `python3 -m http.server` などを実行して
  `http://localhost:8000` を開く
- iPhone / iPad では Safari の「ホーム画面に追加」でフルスクリーンで遊べます

## ゲームの内容

| ワールド | ステージ | しかけ |
|---|---|---|
| 🌼 くさはらワールド | レベル 1〜4 | しばふ・すなば・ぽよんきのこ |
| 🐚 うみべワールド | レベル 5〜8 | みず・ブーストやじるし・ドーナツじま |
| 🌈 おそらワールド | レベル 9〜12 | つるつるこおり・ピンボール・スターパレス |

### たのしい要素

- ひっぱって はなすだけの かんたん操作（スリングショット式・大きな矢印つき）
- 敵に あたると ポン！と星になって はじける（コンボで ほめことばが変わる）
- 全部たおすと 虹のゴールあなが 出現（カービィボウルと同じルール）
- ふわふわ浮かぶ島のコース、流れる雲、にこにこ太陽、揺れるお花
- 水しぶき・紙ふぶき・花火・にじいろの軌跡などのパーティクル演出
- 効果音とBGMは すべて Web Audio で合成（音声ファイルなし・オフラインOK）
- ぷにまるちゃんの表情が変わる（にこにこ・びっくり・めがまわる・うるうる）
- ボタンは大きく、文字は ひらがな中心。まちがえても 悲しい音は しません

## フォルダ構成

```
korokoro-star-golf/
├── index.html            エントリーポイント
├── manifest.webmanifest  ホーム画面追加用
├── css/
│   └── style.css         画面・ボタン・アニメーション
├── js/
│   ├── util.js           共通ヘルパー
│   ├── audio.js          効果音とBGM（WebAudio合成）
│   ├── levels.js         コースデータ（テキストで定義）
│   ├── particles.js      パーティクル演出
│   ├── render.js         Canvas描画（そら・コース・キャラ）
│   ├── game.js           ゲームロジック・物理・入力・UI
│   └── main.js           起動とメインループ
└── assets/
    └── icon.svg          アイコン
```

## コースを増やしたいとき

`js/levels.js` のテキストグリッドを編集するだけです。

```
記号:  . しばふ   s すなば   w みず   i こおり   # しげみ
       b ぽよんきのこ   > < ^ v ブースト
       P スタート   G ゴール   E てき   F フルーツ   (空白) そら
```
